# Technical Data Repo

Bu repo, Product Request Form uygulamasındaki teknik PDF butonları için hazırlanmıştır.

## GitHub'da kurulum

1. GitHub'da yeni public repo oluşturun:
   `technical-data`

2. Bu klasörün içindeki dosyaları repo ana dizinine yükleyin:
   - `index.html`
   - `404.html`
   - `.nojekyll`
   - `technical-sheets/README.txt`

3. Teknik PDF dosyalarını şu klasöre yükleyin:
   `technical-sheets`

4. Repo > Settings > Pages:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
   - Save

5. Yayın adresi şu olur:
   `https://ayturising1-eng.github.io/technical-data/`
